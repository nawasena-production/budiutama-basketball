## 1. Pendahuluan

### 1.1 Tujuan Dokumen

SDD ini menjabarkan keputusan desain teknis sistem Budi Utama Basketball Management System v2.0 yang dibangun dengan Flutter dan Firebase. Dokumen ini menjadi panduan implementasi bagi seluruh anggota tim pengembang.

### 1.2 Prinsip Desain

- **Firebase-First** — manfaatkan fitur bawaan Firebase (offline, real-time, auth) sebelum membuat solusi custom
- **Separation of Concerns** — UI layer (Flutter widgets), business logic (Riverpod providers), dan data layer (Firestore repositories) terpisah dengan jelas
- **Immutability First** — match events tidak pernah dihapus; undo menggunakan event korektif
- **Dual-Write Strategy** — materialized stats untuk kecepatan live; Cloud Functions untuk konsistensi final
- **Readable IDs** — semua document ID menggunakan format deskriptif, bukan UUID
- **Separation of Concerns** — UI layer (Flutter widgets) terpisah dari business logic (Riverpod providers) dan data layer (Firestore repositories)

### 3.3 Kalkulasi Statistik dari player_stats

Persentase dihitung di Flutter client dari materialized stats:

```dart
// lib/core/utils/stats_calculator.dart

class StatsCalculator {
  static double ftPercentage(int made, int attempted) =>
    attempted > 0 ? (made / attempted * 100) : 0.0;

  static double fg2Percentage(int made, int attempted) =>
    attempted > 0 ? (made / attempted * 100) : 0.0;

  static double fg3Percentage(int made, int attempted) =>
    attempted > 0 ? (made / attempted * 100) : 0.0;

  static double fgPercentage(int fg2Made, int fg2Att, int fg3Made, int fg3Att) {
    final totalMade = fg2Made + fg3Made;
    final totalAtt  = fg2Att + fg3Att;
    return totalAtt > 0 ? (totalMade / totalAtt * 100) : 0.0;
  }

  static double zonePercentage(Map<String, int> zone) {
    final attempted = zone['attempted'] ?? 0;
    final made = zone['made'] ?? 0;
    return attempted > 0 ? (made / attempted * 100) : 0.0;
  }
}
```

### 3.4 Zone Classification Logic

```dart
// lib/core/utils/zone_classifier.dart

const _paintX = (0.35, 0.65);
const _paintY = (0.0, 0.30);
const _threePtRadius = 0.43;
const _ringX = 0.5;
const _ringY = 0.08;

String classifyZone(double x, double y) {
  final dist = sqrt(pow(x - _ringX, 2) + pow(y - _ringY, 2));

  if (x >= _paintX.$1 && x <= _paintX.$2 && y <= _paintY.$2) {
    return 'PAINT';
  }
  if (y < 0.18 && dist > _threePtRadius) {
    return x < 0.5 ? 'CORNER_LEFT' : 'CORNER_RIGHT';
  }
  if (dist > _threePtRadius) {
    if (y > 0.55) return 'CENTER_3';
    return x < 0.5 ? 'WING_LEFT' : 'WING_RIGHT';
  }
  if (x < 0.42) return 'MEDIUM_LEFT';
  if (x > 0.58) return 'MEDIUM_RIGHT';
  return 'MEDIUM_CENTER';
}

int calculateDistanceFt(double x, double y) {
  const halfCourtFt = 45.9;
  const courtWidthFt = 49.2;
  final distNorm = sqrt(pow(x - _ringX, 2) + pow(y - _ringY, 2));
  return (distNorm * sqrt(pow(halfCourtFt, 2) + pow(courtWidthFt, 2)) / 1.414).round();
}

bool validateZoneActionConsistency(String actionType, String zone) {
  const zones2pt = {'PAINT', 'MEDIUM_LEFT', 'MEDIUM_CENTER', 'MEDIUM_RIGHT'};
  const zones3pt = {'CORNER_LEFT', 'CORNER_RIGHT', 'WING_LEFT', 'WING_RIGHT', 'CENTER_3'};
  if ((actionType == '2PT_MADE' || actionType == 'MISS_2PT') && zones3pt.contains(zone)) return false;
  if ((actionType == '3PT_MADE' || actionType == 'MISS_3PT') && zones2pt.contains(zone)) return false;
  return true;
}
```

### 3.5 Timer Calculation

```dart
// lib/core/utils/timer_calculator.dart

double currentRemainingSeconds(TimerState state) {
  if (!state.isRunning || state.startedAt == null) {
    return state.secondsAtStart;
  }
  final elapsed = DateTime.now()
    .difference(state.startedAt!.toDate())
    .inMilliseconds / 1000.0;
  return (state.secondsAtStart - elapsed).clamp(0.0, double.infinity);
}
```

## 4. Desain Firebase Security Rules

```javascript
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {

    // ── HELPER FUNCTIONS ──────────────────────────────────────
    function isAuth()         { return request.auth != null; }
    function role()           { return request.auth.token.role; }
    function isManager()      { return isAuth() && role() == 'manager'; }
    function isCoach()        { return isAuth() && role() == 'coach'; }
    function isStatistician() { return isAuth() && role() == 'statistician'; }
    function isPlayer()       { return isAuth() && role() == 'player'; }
    function isStaff()        { return isManager() || isCoach(); }

    function matchIsActive(matchData) {
      return matchData.current_state in
        ['Q1_ACTIVE','Q2_ACTIVE','Q3_ACTIVE','Q4_ACTIVE','OT_ACTIVE'];
    }

    // ── USERS ─────────────────────────────────────────────────
    match /users/{userId} {
      allow read:  if isAuth() && (isStaff() || request.auth.uid == resource.data.uid);
      allow write: if false; // hanya Cloud Functions yang bisa write
    }

    // ── TEAMS ─────────────────────────────────────────────────
    match /teams/{teamId} {
      allow read:  if isAuth();
      allow write: if isManager();
    }

    // ── PLAYERS ───────────────────────────────────────────────
    match /players/{playerId} {
      allow read: if isAuth() && (
        isStaff() || isStatistician() ||
        (isPlayer() && request.auth.uid == resource.data.user_id)
      );
      allow create: if isManager();
      allow update: if isManager();
      allow delete: if false; // soft delete only
    }

    // ── EVENTS (TURNAMEN) ─────────────────────────────────────
    match /events/{eventId} {
      allow read:  if isAuth();
      allow write: if isManager();
    }

    // ── MATCHES ───────────────────────────────────────────────
    match /matches/{matchId} {
      allow read: if isAuth();
      allow create: if isManager();
      allow update: if isManager() || isStatistician();
      allow delete: if false;

      // TIMER STATE — hanya Statistician yang bisa update
      match /timer_state/{doc} {
        allow read:   if isAuth();
        allow write:  if isStatistician();
      }

      // EVENTS — immutable setelah dibuat
      // Pengecualian: is_undone bisa diupdate oleh Statistician untuk undo
      match /events/{eventId} {
        allow read:   if isAuth();
        allow create: if isStatistician();
        allow update: if isStatistician()
          && request.resource.data.diff(resource.data).affectedKeys()
             .hasOnly(['is_undone']);  // hanya field is_undone yang bisa diupdate

      // PLAYER STATS — Statistician bisa update saat match aktif
      match /player_stats/{playerId} {
        allow read:   if isAuth();
        allow write:  if isStatistician();
      }

      // LINEUPS — Statistician bisa update
      match /lineups/{playerId} {
        allow read:   if isAuth();
        allow write:  if isStatistician();
      }
    }

    // ── INJURY REPORTS ────────────────────────────────────────
    match /injury_reports/{reportId} {
      allow read:  if isStaff();
      allow write: if isStaff();
    }

    // ── TRAINING SESSIONS ─────────────────────────────────────
    match /training_sessions/{sessionId} {
      allow read:  if isAuth();
      allow write: if isManager();
    }

    // ── PHYSICAL TEST SESSIONS ────────────────────────────────

      allow write: if isStaff();

      match /results/{playerId} {
        allow read:  if isStaff();
        allow write: if isStaff();
      }

      allow read:  if isStaff();
```

### 6.2 Match Action Notifier

```dart
@riverpod
class MatchActionNotifier extends _$MatchActionNotifier {

  Future<void> recordAction({
    required String matchId,
    required String playerId,
    required String actionType,
    required int quarter,
    required double timeRemaining,
    String? zone,
    double? shotX,
    double? shotY,
    int? shotDistanceFt,
    bool isOpponent = false,
  }) async {
    final firestore = FirebaseFirestore.instance;

    // Generate event ID: q{quarter}_{sequence}
    final seq = await _getNextSequence(matchId, quarter);
    final eventId = 'q${quarter}_${seq.toString().padLeft(3, '0')}';

    // Kalkulasi value dari action type
    final value = _valueFromAction(actionType);

    // Batch write: event + player_stats update
    final batch = firestore.batch();

    // 1. Tambah event baru (immutable)
    batch.set(
      firestore.collection('matches/$matchId/events').doc(eventId),
      {
        'quarter': quarter,
        'time_remaining': timeRemaining,
        'player_id': isOpponent ? null : playerId,
        'action_type': actionType,
        'value': value,
        'zone': zone,
        'shot_x': shotX,
        'shot_y': shotY,
        'shot_distance_ft': shotDistanceFt,
        'is_opponent': isOpponent,
        'is_undone': false,
        'undo_ref_id': null,
        'created_by': ref.read(currentUserProvider).id,
        'created_at': FieldValue.serverTimestamp(),
      },
    );

    // 2. Update materialized player_stats (hanya untuk tim sendiri)
    if (!isOpponent) {
      final statsRef = firestore
        .collection('matches/$matchId/player_stats')
        .doc(playerId.split('_putra').first.split('_putri').first);  // jersey_inisial
      batch.update(statsRef, _buildStatsIncrement(actionType, zone));
    }

    // 3. Update match score
    if (!isOpponent && value > 0) {
      batch.update(
        firestore.collection('matches').doc(matchId),
        {'home_score': FieldValue.increment(value)},
      );
    } else if (isOpponent && value > 0) {
      batch.update(
        firestore.collection('matches').doc(matchId),
        {'opponent_score': FieldValue.increment(value)},
      );
    }

    await batch.commit();
  }

  Future<void> undoLastAction(String matchId) async {
    final firestore = FirebaseFirestore.instance;

    // Cari event terakhir yang belum di-undo
    final lastEvent = await firestore
      .collection('matches/$matchId/events')
      .where('is_undone', isEqualTo: false)
      .where('action_type', whereNotIn: ['UNDO', 'STATE_TRANSITION', 'TIMER_START', 'TIMER_PAUSE', 'TIMER_RESUME'])
      .orderBy('created_at', descending: true)
      .limit(1)
      .get();

    if (lastEvent.docs.isEmpty) return;

    final eventToUndo = lastEvent.docs.first;
    final eventData = eventToUndo.data();

    final batch = firestore.batch();

    // Mark event sebagai undone
    batch.update(eventToUndo.reference, {'is_undone': true});

    // Tambah UNDO event
    final undoSeq = await _getNextSequence(matchId, eventData['quarter']);
    final undoId = 'q${eventData['quarter']}_${undoSeq.toString().padLeft(3, '0')}';
    batch.set(
      firestore.collection('matches/$matchId/events').doc(undoId),
      {
        'quarter': eventData['quarter'],
        'time_remaining': eventData['time_remaining'],
        'action_type': 'UNDO',
        'undo_ref_id': eventToUndo.id,
        'is_undone': false,
        'created_by': ref.read(currentUserProvider).id,
        'created_at': FieldValue.serverTimestamp(),
      },
    );

    // Reverse stats update (decrement)
    if (!eventData['is_opponent']) {
      final playerId = eventData['player_id'] as String;
      final statsRef = firestore
        .collection('matches/$matchId/player_stats')
        .doc(playerId.split('_putra').first.split('_putri').first);
      batch.update(statsRef, _buildStatsDecrement(
        eventData['action_type'],
        eventData['zone'],
      ));
    }

    // Reverse score
    final value = eventData['value'] as int? ?? 0;
    if (value > 0) {
      final scoreField = eventData['is_opponent'] ? 'opponent_score' : 'home_score';
      batch.update(
        firestore.collection('matches').doc(matchId),
        {scoreField: FieldValue.increment(-value)},
      );
    }

    await batch.commit();
  }

  Map<String, dynamic> _buildStatsIncrement(String actionType, String? zone) {
    final increments = <String, dynamic>{};
    switch (actionType) {
      case '1PT_MADE':
        increments['ft_made'] = FieldValue.increment(1);
        increments['ft_attempted'] = FieldValue.increment(1);
        increments['points'] = FieldValue.increment(1);
      case '2PT_MADE':
        increments['fg2_made'] = FieldValue.increment(1);
        increments['fg2_attempted'] = FieldValue.increment(1);
        increments['points'] = FieldValue.increment(2);
        if (zone != null) {
          increments['shot_zones.$zone.made'] = FieldValue.increment(1);
          increments['shot_zones.$zone.attempted'] = FieldValue.increment(1);
        }
      case '3PT_MADE':
        increments['fg3_made'] = FieldValue.increment(1);
        increments['fg3_attempted'] = FieldValue.increment(1);
        increments['points'] = FieldValue.increment(3);
        if (zone != null) {
          increments['shot_zones.$zone.made'] = FieldValue.increment(1);
          increments['shot_zones.$zone.attempted'] = FieldValue.increment(1);
        }
      case 'MISS_1PT':
        increments['ft_attempted'] = FieldValue.increment(1);
      case 'MISS_2PT':
        increments['fg2_attempted'] = FieldValue.increment(1);
        if (zone != null) {
          increments['shot_zones.$zone.attempted'] = FieldValue.increment(1);
        }
      case 'MISS_3PT':
        increments['fg3_attempted'] = FieldValue.increment(1);
        if (zone != null) {
          increments['shot_zones.$zone.attempted'] = FieldValue.increment(1);
        }
      case 'ASSIST':      increments['assists'] = FieldValue.increment(1);
      case 'REBOUND_OFF': increments['offensive_rebounds'] = FieldValue.increment(1);
      case 'REBOUND_DEF': increments['defensive_rebounds'] = FieldValue.increment(1);
      case 'STEAL':       increments['steals'] = FieldValue.increment(1);
      case 'TURNOVER':    increments['turnovers'] = FieldValue.increment(1);
      case 'BLOCK':       increments['blocks'] = FieldValue.increment(1);
      case 'FOUL':        increments['fouls'] = FieldValue.increment(1);
    }
    increments['updated_at'] = FieldValue.serverTimestamp();
    return increments;
  }
}
```

```typescript
// functions/src/index.ts

import * as functions from 'firebase-functions';
import * as admin from 'firebase-admin';

admin.initializeApp();
const db = admin.firestore();

// ── TRIGGER: Pertandingan selesai (POST_MATCH) ─────────────────
export const onMatchFinished = functions.firestore
  .document('matches/{matchId}')
  .onUpdate(async (change, context) => {
    const before = change.before.data();
    const after  = change.after.data();
    const matchId = context.params.matchId;

    // Hanya trigger saat transisi ke POST_MATCH
    if (before.current_state === after.current_state) return;
    if (after.current_state !== 'POST_MATCH') return;

    // Ambil semua events untuk rekap final
    const eventsSnap = await db
      .collection(`matches/${matchId}/events`)
      .where('is_undone', '==', false)
      .get();

    // Hitung statistik final dari event log (verifikasi vs materialized stats)
    const finalStats = calculateFinalStats(eventsSnap.docs);

    // Update player_stats dengan nilai final yang terverifikasi
    const batch = db.batch();
    for (const [playerId, stats] of Object.entries(finalStats)) {
      batch.update(
        db.collection(`matches/${matchId}/player_stats`).doc(playerId),
        { ...stats, final_calculated: true }
      );
    }
    await batch.commit();
  });

// ── TRIGGER: Event baru dibuat → Audit Log ─────────────────────
export const onMatchEventCreated = functions.firestore
  .document('matches/{matchId}/events/{eventId}')
  .onCreate(async (snap, context) => {
    const event = snap.data();
    await db.collection('audit_logs').add({
      user_id:     event.created_by,
      action_type: 'MATCH_EVENT_CREATED',
      entity_type: 'match_event',
      entity_id:   context.params.eventId,
      new_value:   { match_id: context.params.matchId, action_type: event.action_type },
      created_at:  admin.firestore.FieldValue.serverTimestamp(),
    });
  });

// ── TRIGGER: Injury status berubah → Update player status ──────
export const onInjuryStatusChanged = functions.firestore
  .document('injury_reports/{reportId}')
  .onUpdate(async (change) => {
    const after = change.after.data();
    if (after.status === 'cleared') {
      await db.collection('players').doc(after.player_id).update({
        status: 'active',
        updated_at: admin.firestore.FieldValue.serverTimestamp(),
      });
    }
  });

// ── HELPER: Kalkulasi statistik final ─────────────────────────
function calculateFinalStats(
  eventDocs: admin.firestore.QueryDocumentSnapshot[]
): Record<string, object> {
  const stats: Record<string, any> = {};

  for (const doc of eventDocs) {
    const e = doc.data();
    if (e.is_opponent || !e.player_id) continue;

    const key = e.player_id.split('_putra')[0].split('_putri')[0];
    if (!stats[key]) stats[key] = initPlayerStats();

    switch (e.action_type) {
      case '1PT_MADE':   stats[key].ft_made++;  stats[key].ft_attempted++;  stats[key].points += 1; break;
      case '2PT_MADE':   stats[key].fg2_made++; stats[key].fg2_attempted++; stats[key].points += 2; break;
      case '3PT_MADE':   stats[key].fg3_made++; stats[key].fg3_attempted++; stats[key].points += 3; break;
      case 'MISS_1PT':   stats[key].ft_attempted++;  break;
      case 'MISS_2PT':   stats[key].fg2_attempted++; break;
      case 'MISS_3PT':   stats[key].fg3_attempted++; break;
      case 'ASSIST':     stats[key].assists++;        break;
      case 'REBOUND_OFF':stats[key].offensive_rebounds++; break;
      case 'REBOUND_DEF':stats[key].defensive_rebounds++; break;
      case 'STEAL':      stats[key].steals++;         break;
      case 'TURNOVER':   stats[key].turnovers++;      break;
      case 'BLOCK':      stats[key].blocks++;         break;
      case 'FOUL':       stats[key].fouls++;          break;
    }
  }
  return stats;
}
```

## 11. Desain Deployment

### 11.1 Environments

Sistem menggunakan **1 Firebase project** (`budiutama-basketball`) untuk semua fase. Isolasi development dilakukan via **Firebase Emulator Suite** yang berjalan lokal di mesin developer — tidak menyentuh project Firebase sama sekali saat coding sehari-hari.

| Environment | Branch | Firebase Project | Keterangan |
|---|---|---|---|
| Development | `feature/*` | Firebase Emulator (lokal) | Developer coding; tidak menyentuh project Firebase |
| Testing / Staging | `develop` | `budiutama-basketball` | Testing integrasi di Firebase nyata; data boleh kotor |
| Production | `main` | `budiutama-basketball` | Data production; clear dulu sebelum go-live pertama |

**Alur clear data sebelum go-live:**
```bash
# 1. Backup data testing (opsional, untuk referensi)
gcloud firestore export gs://budiutama-basketball.appspot.com/backup-pre-prod

# 2. Hapus semua data testing dari Firestore
firebase firestore:delete --all-collections --project budiutama-basketball

# 3. Seed data production awal (tim, akun pengguna)
dart run scripts/clear_and_seed.dart
```

Catatan: Security Rules, Firestore Indexes, dan Cloud Functions tidak ikut terhapus — hanya data dokumen Firestore yang dihapus.

### 11.2 Firebase Emulator — Development Lokal

```dart
// main.dart — developer mode
// kDebugMode = true saat flutter run, false saat flutter build --release

if (kDebugMode) {
  // Arahkan ke emulator lokal — tidak menyentuh Firebase production
  await FirebaseFirestore.instance
      .useFirestoreEmulator('localhost', 8080);
  await FirebaseAuth.instance
      .useAuthEmulator('localhost', 9099);
  FirebaseFunctions.instance
      .useFunctionsEmulator('localhost', 5001);
}
// Release build otomatis konek ke Firebase production
```

```bash
# Jalankan emulator sebelum flutter run
firebase emulators:start

# Emulator yang aktif:
# - Firestore  : localhost:8080
# - Auth       : localhost:9099
# - Functions  : localhost:5001
# - Hosting    : localhost:5000
# - Emulator UI: localhost:4000
```

### 11.3 CI/CD Pipeline (GitHub Actions)

```yaml
# Setiap push (semua branch):
- Flutter analyze (static analysis)
- Flutter test (unit + widget tests)
- Firebase Rules unit tests (via Emulator)

# Push ke develop:
- Build Flutter Web
- Deploy Security Rules ke budiutama-basketball
- Deploy Cloud Functions ke budiutama-basketball
- Deploy Flutter Web ke Firebase Hosting (preview channel)

# Push ke main (dengan manual approval):
- Build Flutter APK release (signed)
- Build Flutter Web release
- Deploy Security Rules ke budiutama-basketball
- Deploy Cloud Functions ke budiutama-basketball
- Deploy Flutter Web ke Firebase Hosting (live channel)
- Upload APK ke Play Store internal track
```

### 11.3 Flutter Build Targets

| Platform | Command | Output |
|---|---|---|
| Android Debug | `flutter build apk --debug` | `.apk` untuk testing |
| Android Release | `flutter build apk --release` | `.apk` + Play Store bundle |
| iOS Release | `flutter build ios --release` | `.ipa` untuk App Store |
| Web | `flutter build web --release` | Static files → Firebase Hosting |

### 11.4 Firebase CLI Deployment

```bash
# Deploy semua Firebase services
firebase deploy

# Deploy hanya Security Rules
firebase deploy --only firestore:rules

# Deploy hanya Cloud Functions
firebase deploy --only functions

# Deploy hanya Flutter Web ke Hosting
firebase deploy --only hosting
```

### 11.5 Environment Variables & Secrets

| Konfigurasi | Lokasi | Keterangan |
|---|---|---|
| `google-services.json` | `android/app/` — tidak di-commit | Firebase Android config |
| `GoogleService-Info.plist` | `ios/Runner/` — tidak di-commit | Firebase iOS config |
| `firebase_options.dart` | Generated by FlutterFire CLI | Berisi API keys — tambahkan ke `.gitignore` |
| Cloud Functions env | Firebase Functions config | Secret keys untuk admin operations |
